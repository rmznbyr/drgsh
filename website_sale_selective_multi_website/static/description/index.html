Add Multiple Websites per product in Odoo multi website

Get the best of your Multiwebsite instance with oopo's Website Sale Selective Multi Website. Do more, in less time!

Works out of the box, with this app you can add a product to several websites instead of just one or all at the time.

Odoo's core multi website feature only allows you to select one website per product or all at, but it doesn't allow you to select 2 or more websites per product.

Stay Ahead Of Competition

Get the best of your multiwebsite instance

Opoo's Website Sale Selective Multi Website

With this app you will be able to:

  • Add a single website to a product
  • Add 2 or more websites to a product
  • Add All websites to a product

To start using Oppo's Website Sale Selective Multi Website install the App and then go to Website - Products - Product Templates and go to theEcommerce Tab and add the websites you need to your product or leave empty to add the product to all the websites

Website 1 shows the product Acoustic Bloc Screens

Website 2 DOES NOT show the product Acoustic Bloc Screens
